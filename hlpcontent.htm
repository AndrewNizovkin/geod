Geod это возможность легко решать простые задачи

   GEOD применяется для автоматизации камеральной обработки полевых измерений при создании планово-высотных геодезических сетей и производстве топографических съёмок.

Функциональные возможности программы:

  • Импорт полевых данных из электронных тахеометров 3ТА5Р, Leica, Sokkia, Topcon, Nicon, Trimble;
  • Редактирование и обработка тахеометрической съёмки для использования в ГИС: Autocad, Credo, Mapinfo, Digitals;
  • Уравнивание теодолитных ходов с различными способами привязки к опорной геодезической сети;
  • Решение задачи съёмки шельфа методом прямых угловых засечек;
  • Создание отчётов для печати на принтере;
  • Решение некоторых прикладных задач геодезии.
